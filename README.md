# 治療日誌マスタ化出力ツール

治療日誌の `.xlsm` を複数読み込みし、`ThisWorkbook` 側に縦持ちマスタを生成する VBA ソースです。

## 構成

- [src/modTreatmentDiaryImporter.bas](C:/Users/yuuki/OneDrive/ドキュメント/マイドキュメント/システム案件/治療日誌マスタ化出力ツール/src/modTreatmentDiaryImporter.bas)
- [dist/modTreatmentDiaryImporter_sjis.bas](C:/Users/yuuki/OneDrive/ドキュメント/マイドキュメント/システム案件/治療日誌マスタ化出力ツール/dist/modTreatmentDiaryImporter_sjis.bas)

## 前提

- 出力先はこの VBA を格納したブックです。
- 取り込み対象シートは `～10日` `11～20日` `21日～` の3枚です。
- 取り込み元ブックには `転記` シートがあり、`B35` に1ファイルの合計患者数が入っている前提です。
- 対象データは `A1:DF53` を固定範囲で読み取ります。
- ブロック開始列は `A, K, U, AE, AO, AY, BI, BS, CC, CM, CW` です。
- フルパスから最後のファイル名部分を切り出し、さらにファイル名内に `／` や `/` が含まれる場合はその後ろを論理ファイル名として扱います。例: `治療日誌2025年08月／04学芸大.xlsm` -> `04学芸大.xlsm` -> `04`
- `店舗リファレンス` の `使用中フラグ` は空欄を有効扱いにしています。`0 / FALSE / NO / N / 無効 / 停止` は無効扱いです。
- 同一ファイル内に複数 `yyyymm` が混在した場合は致命エラーにしています。

## セットアップ

1. 任意のマクロ有効ブックを作成します。
2. VBA エディタで標準モジュールを追加し、[dist/modTreatmentDiaryImporter_sjis.bas](C:/Users/yuuki/OneDrive/ドキュメント/マイドキュメント/システム案件/治療日誌マスタ化出力ツール/dist/modTreatmentDiaryImporter_sjis.bas) をインポートします。
3. `InitializeTreatmentDiaryWorkbook` を実行します。
4. 生成された `店舗リファレンス` シートに `店舗ID / 店舗名 / 使用中フラグ` を設定します。

## 実行マクロ

- `InitializeTreatmentDiaryWorkbook`
  - 必要シートとヘッダーを作成します。
- `ImportTreatmentDiariesFromFolder`
  - 指定フォルダ配下の `.xlsm` を再帰的に取り込みます。
- `ImportTreatmentDiariesFromFiles`
  - 選択した複数 `.xlsm` を取り込みます。

## 取込時の注意

- VBA へ直接インポートする場合は Shift-JIS 版の [dist/modTreatmentDiaryImporter_sjis.bas](C:/Users/yuuki/OneDrive/ドキュメント/マイドキュメント/システム案件/治療日誌マスタ化出力ツール/dist/modTreatmentDiaryImporter_sjis.bas) を使ってください。
- UTF-8 版をそのままインポートすると、日本語のシート名やメッセージ定数が文字化けし、`インデックスが有効範囲にありません。` のような切り分けしづらいエラーになることがあります。

## 出力シート

- `治療日誌マスタ`
  - `日付, 年月, 施術時間, 分類, 担当, 患者名, 料金合計, 店舗ID, 店舗名, 元ファイル名, 元ファイルパス, 元シート名, 取込日時`
- `取込実行ログ`
  - `実行日時, ファイル名, シート名, ステータス, 読取件数, 出力件数, メッセージ`
- `取込詳細ログ`
  - `実行日時, ファイル名, シート名, 日付, 患者名, 担当, 種別, 内容`

## 実装済みルール

- `施術時間` が空でない行のみ取込対象です。
- `患者名` は前後空白を除去します。
- 料金は `+4` から `+9` を合算します。
- 料金セルが文字列またはエラー値の場合は `0` 扱いで詳細ログに警告を残します。
- `担当空欄` と `患者名空欄` は警告ログを出しつつ処理継続します。
- `日付取得不可` `対象シートなし` `複数年月混在` `重複取込` はそのファイルを中止します。
- `転記!B35` と出力件数が一致しない場合はそのファイルを中止します。
- 再取込判定は論理ファイル名と `年月` で行います。例: `治療日誌2025年08月／04学芸大.xlsm` は `04学芸大.xlsm` として判定します。
